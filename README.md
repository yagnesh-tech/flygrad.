# FlyGrad

## Overview
FlyGrad is a web-based student support platform developed to provide useful information for students in one place. The project focuses on important areas of student life such as housing, part-time jobs, placement packages, login access, and general project information. It is designed with a simple and user-friendly interface so that students can easily navigate through different sections.

## Objectives
The main objectives of FlyGrad are:
- To provide students with information related to housing options
- To help students explore part-time job opportunities
- To create awareness about placement packages and career opportunities
- To offer a simple web-based platform for student use
- To improve accessibility of student-related resources

## Features
- Home page for easy navigation
- About page describing the project
- Contact page for communication details
- Login page for user authentication interface
- Housing page for accommodation-related details
- Part-time jobs page for job-related information
- Placement packages page for career and salary insights

## Project Structure
The project consists of the following files:

- `index.html` – Home page
- `about.html` – About FlyGrad
- `contact.html` – Contact page
- `login.html` – Login page
- `housing.html` – Housing information
- `part-time-jobs.html` – Part-time jobs information
- `placement-packages.html` – Placement package details

## Technologies Used
- HTML
- CSS
- JavaScript

## Working of the Project
FlyGrad works as a static website where users can open different pages and access information based on their needs. The homepage acts as the main entry point, from where users can navigate to pages related to jobs, housing, placements, and other details. The project is mainly intended to demonstrate front-end web development concepts and student-oriented website design.

## Advantages
- Simple and easy to use
- Provides multiple student resources in one platform
- Improves awareness of career and job opportunities
- Helps students access useful information quickly
- Can be expanded in future with advanced features

## Future Enhancements

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any web browser

## Conclusion
FlyGrad is a useful student-centered web project that combines academic and career-related support features into a single platform. It serves as a good example of a beginner-friendly web development project and can be further enhanced into a complete student service portal.
